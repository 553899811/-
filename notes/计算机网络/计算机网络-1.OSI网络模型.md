# 网络层次划分
## 1.常用网络层次划分
### 1.1 OSI/RM模型
```
   该模型将计算机网络体系结构的通信协议划分为7层,自下而上为:
       物理层(Physics Layer)
       数据链路层(Data Link Layer)
       网络层(Network Layer)
       传输层(Transport Layer)
       会话层(Session Layer)
       表现层(Presentation Layer)
       应用层(Application Layer)
```
