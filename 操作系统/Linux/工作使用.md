# Linux常见工作使用命令
## 内存篇
 - 查看内存使用情况
```
  free -m 显示结果以MB为单位 
  free    显示结果以KB为单位
  free -g 显示结果以GB为单位
```
  - 查看占用内存最大的进程,顺序排序
```
  ps aux|head -1;ps aux|grep -v PID|sort -rn -k +4|head
```
## 进程篇
 - 查看某一个进程占用多少线程
 ```
    ps hH p <pid> |wc -l
 ```
 - 根据名称用ps命令查看进程ID
 ```
   ps -ef|grep tomcat
 ```
 - 查看到进程ID之后,使用netstat命令查看其占用的端口:
 ```
    netstat -anp|grep 10997
 ```
